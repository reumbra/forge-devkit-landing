### Document Conventions
- **File naming:** `NNN-topic-name.md` for chronological entries

- **Agent-friendly format:** Every doc starts with `> **Context:**` one-liner, then structured data
- **Decisions:** Marked with `DECISION:` prefix in bold
- **Open questions:** Marked with `OPEN:` prefix
- **Principles:** Extracted to PRINCIPLES.md with source reference
