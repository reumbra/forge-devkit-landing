### Key Decisions (v2)

| Decision | Date | Rationale |
|----------|------|-----------|
| Developer-only landing | 2026-03-06 | Entrepreneurs get separate landing later. Dev focus = sharper messaging |
| Arc B: "From Vibe Code to Production" | 2026-03-06 | Covers 3/4 JTBD segments, works with Confirmation Bias, widest Loss Aversion framing |
| Module Hub architecture (10 pages) | 2026-03-06 | 4 module pages (core, product, qa, autopilot) + overview + pricing + 3 comparison |
| forge-autopilot as peak moment | 2026-03-06 | Peak-End Rule: wow factor in middle, not hero. Hero = broad pain recognition |
| tracker/worktree = cards only | 2026-03-06 | Pareto: weak standalone selling points, appear in ecosystem sections |
| One-time pricing positioning | 2026-03-06 | Door-in-the-Face vs Devin $20/mo ($240/yr). "One purchase, artifacts stay forever" |
